## 🧠 About LearnDNA AI

**LearnDNA AI** is an intelligent learning platform designed to transform the way students study.

Traditional learning can feel repetitive and boring. LearnDNA AI makes studying **interactive, competitive, and engaging** by combining:

✨ Artificial Intelligence  
🔥 Gamification  
📚 Smart Study Tools  
📈 Performance Tracking  

This project was built as a **Final Year Student Project** using Flutter and Firebase.

---

## 🎯 Problem Statement

Students often struggle with:

- Low motivation while studying
- Lack of personalized learning support
- Poor progress tracking
- No engaging revision tools
- Limited gamified learning experiences

LearnDNA AI solves these challenges through an AI-powered educational ecosystem.

## 💡 Core Features

### 🔐 Authentication System
Secure login and signup with Firebase Authentication.

### 👤 Smart Profile System
Users can:
- Create accounts
- Set display names
- Upload profile avatars
- Track learning identity

### 🤖 AI Learning Experience
AI-powered educational assistance for smarter studying.

### 🧠 Quiz Engine
Interactive quizzes for knowledge testing.

Features:
- Multiple questions
- Score tracking
- Instant feedback
- XP rewards

### 🏆 Gamification System
Learning becomes addictive with:

- XP Rewards
- Leaderboard Rankings
- Daily Streak Tracking
- Achievement Motivation

### 📚 Flashcard Revision
Quick memory-based revision system for students.

### 📅 Study Planner
Helps learners organize daily academic goals.

### 🎓 Certificate Generation
Reward-based completion certificates.

### 🌙 Dark Mode
Modern UI with light/dark theme switching.

### 📊 Analytics Dashboard
Track student learning performance visually.
